① 컨트롤러 설치
  • 컨트롤러에 건전지를 삽입합니다.
  • 에어컨 제조사에 맞는 프로토콜을 선택합니다.

  • 컨트롤러를 에어컨 가까이 가져가 블루투스 기반으로 제어 테스트를 진행합니다. (이 단계에서는 허브 등록 없이 단독으로 테스트합니다.)
  • 제어가 정상 확인되면, 아래 부착 기준에 따라 IR 수신부 방향으로 컨트롤러를 부착합니다.
    • 1. IR 수신부와 평행이 되도록 설치합니다.
    • 2. 필터 위에는 설치하지 않습니다.
    • 3. 실리콘을 적당량 도포합니다.
    • 4. 낙하되지 않도록 견고하게 고정합니다.
  • 부착 완료 후 제어 테스트를 한 번 더 진행하여 정상 동작을 최종 확인합니다.

도면 작성 예시
1F 1H / 1C~4C
2F 5C~7C ← 이어서
H 허브
C 컨트롤러
S 센서
A 공유기
⚠ 번호는 층이 바뀌어도 이어서 증가합니다. 1층 마지막 컨트롤러가 4C라면 2층 첫 번째는 5C부터 시작합니다. 허브도 1층 1H → 2층 2H로 이어집니다.
도면 등록 방법
  • 파트너스앱 [장소 보기]에서 오른쪽 상단 [도면] 버튼을 누릅니다.
    • 1. [+]를 누른 후 카메라 표시를 눌러 도면을 등록합니다.
    • 2. 도면은 층별로 1개씩만 등록 가능하며, 모든 층마다 빠짐없이 등록합니다.

기기 표기 방법
  • 허브 — 번호 형식: [순번+H] (예: 1H → 2H → 3H)
    • 1. 층이 바뀌어도 번호는 이어서 증가합니다.
    • 2. 허브 MAC 주소 12자리를 함께 기재합니다.
  • 공유기 — 번호 형식: [층+알파벳] (예: 1층 2대 → 1A, 1B / 2층 1대 → 2A)
  • 컨트롤러·센서 — 번호 형식: [순번+C] / [순번+S] (예: 1C, 2C / 1S, 2S)
    • 1. 층이 바뀌어도 번호는 이어서 증가합니다.
    • 2. 고객이 주로 사용하는 에어컨에 앞번호를 부여합니다.
    • 3. 사용 빈도가 낮은 기기는 후순위로 배정합니다.
    • 4. MAC 주소 12자리를 모두 기재하고, MAC 주소 옆 칸에 냉난방기 종류를 기재합니다. (예: 1way 스탠드 / 4way 천장형)
도면 작성 규칙
  • 1. 중요도 순으로 번호를 부여합니다. 고객이 주로 사용하는 에어컨에 앞번호, 사용 빈도가 낮거나 비상주 공간의 기기는 끝번호로 부여합니다.
  • 2. 파트너스 웹 등록 시, 반드시 [도면 기입 번호] + [공간명(컨트롤러명)] 조합으로 등록합니다.
  • 3. 복잡한 명칭 대신 해당 기기의 번호만 명확히 기입합니다.
도면 작성 주의사항
  • 1. 냉난방기 형태를 기재합니다. 실제 현장과 흡사한 도면 작성을 위해 에어컨 형태를 표기합니다. (예: 1way, 4way, 360형)
  • 2. 기둥, 장애물 등이 있을 경우 도면에 함께 표기합니다.
  • 3. 방위(방향)를 전 층 동일한 기준으로 통일합니다.
    • 단층 매장: 출입문 기준으로 방향을 설정합니다.
    • 복층 매장: 계단 기준으로 방향을 설정합니다.
⚠ 층별 기준을 혼용하지 않습니다. 예를 들어 1층은 출입문 기준, 2층은 계단 입구 기준으로 각각 다르게 설정하면 안 됩니다. 전 층 동일한 기준을 사용합니다.
